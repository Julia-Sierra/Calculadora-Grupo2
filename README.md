# Calculadora Grupo 2

Este proyecto es una calculadora simple en C#.

## Requisitos

- .NET SDK instalado
- Visual Studio Code o Visual Studio

## Cómo ejecutar el proyecto

### Opción 1: desde la terminal

1. Abre la terminal en la carpeta del proyecto.
2. Ejecuta el siguiente comando:

```bash
dotnet run
```

### Opción 2: desde Visual Studio Code

1. Abre la carpeta del proyecto en VS Code.
2. Presiona F5 o usa la opción "Run and Debug".
3. Selecciona la configuración de C# si se te solicita.

## Nota

Si recibes errores al ejecutar, asegúrate de tener instalado el SDK de .NET compatible con este proyecto.
